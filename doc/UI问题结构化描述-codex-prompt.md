## 模板
```text
请根据以下 UI 调整需求，修改对应的代码。只输出修改部分的代码和文件名，并简要说明修改了哪里。

### 问题定位
- 页面/组件：哪个页面或组件（如“首页 Hero 区”、“博客详情页正文”）
- 当前表现：用文字描述现在的样子（可附截图或标注）
- 期望效果：用文字描述你想要的样子
- 相关技术栈：Tailwind CSS / 自定义 CSS / 动画库等
- 影响范围：是否只改这一个地方，还是全局样式？
```
